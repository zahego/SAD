Homework 4: Display Data

Data Model

The data model contain 4 table, 1 user table, 1 other table, 1 associative table and 1 user role table. My web data model is based on an online game website data model so the user table will be replaced with mmo_player table, the other table becomes game_class, associative table becomes your_character table and user role become user_role. The model is based on a many to many relationship between mmo_player table and game_class table and a one to one relationship between mmo_player table and user_role able. The premiere of the database is that one player can hold many game class and one game class can be played by many player. The user will also have a role assigned to them in the context of operating the game. The mmo_player tale holds information about the players and the game_class table holds the information about fantasy class that the player can choose from. The your_character table is the connection between the mmo_player and the game_class and hold the record for each player's game_class record. The user_role table hold information about role of player ranging from player to admin
